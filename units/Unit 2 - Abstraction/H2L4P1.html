Robot in een Doolhof

In dit project, ga je een robot programmeren om door verschillende doolhoven heen te komen.

Er zijn 5 doolhoven voor je gemaakt. De code die je gaat schrijven om te ontsnappen uit deze doolhoven moet zo simpel, beknopt, en elegant mogelijk zijn.

Bijvoorbeeld, deze oplossing zou werken voor doolhof 1:

Maar een herhaal blok nesten binnenin een andere, zorgt ervoor dat de code beknopter wordt en, zoals de meeste programmeurs zouden vinden, ook eleganter:

Dit werkt omdat wanneer de robot zijn bestemming bereikt, het ons niet uitmaakt naar welke kant hij kijkt.

In pseudocode zou je dit kunnen weergeven met MOVE_FORWARD(), ROTATE_RIGHT() and ROTATE_LEFT().


  1. Bestudeer beide de robot en de bord sprites. Schrijf voor elk gegeven doolhoven, de kortste en meest elegante code om de robot te helpen ontsnappen. Gebruik enkel deze 4 speciale Beweging blokken (naast welke Controle blokken dan ook die je nodig hebt):




  2. Analyseer de code. Op blokjes papier, teken het doolhof waarvoor deze code bedoelt was. Gebruik Snap! niet.
  3. Deel je tekening met andere studenten, en bespreek hoe je dit gemaakt hebt.
