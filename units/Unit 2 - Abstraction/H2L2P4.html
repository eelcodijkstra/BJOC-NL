Debugging met

Je kunt pauzeer alles gebruiken, om pauzes in je code te plaatsen zodat je kunt zien wat er gaande weg gebeurd. In deze animatie, runt de programmeur zijn/haar code met pauzeer alles binninin, en klikt op de Run/Pauze knop naast de groene vlag ( /) om door de code heen te bewegen tot een bepaald pauze blok weer is bereikt. In dit voorbeeld, runt er maar één script, maar pauzeer alles zal elk script pauzeren wat aan het runnen is totdat de gele Run/Pauze knop wordt ingedrukt.
