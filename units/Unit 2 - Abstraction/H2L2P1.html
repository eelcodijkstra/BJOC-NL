Ja, jijkunt jouw blokken laten zien welk type data ze verwachten als input: een nummer, een lijst, of een ander type. Sommige talen vereisen dat het data type aangegeven moet zijn. In Snap! is dit een optie. Het is niet perse nodig, maar, net als dat je een kleur toewijst aan een blok, kan het een handige herinnering zijn van wat het blok nu eigenlijk doet en welk type input het verwacht. Je hebt al input ingangen gezien van verschillende vormen en soorten, die elk verschillende datatypen verwachten.

    }
  }
