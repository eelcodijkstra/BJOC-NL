Sprite Die een Sprite Volgt

Op deze pagina, ga je het script aanpassen zodat de sprites niet voor altijd blijven bewegen, maar alleen herhalen tot ze elkaar aanraken.

  1. Pas de code van Sprite(2) aan zodat hij naar Sprite wijst totdat hij Sprite aanraakt. (De instructies en animatie hieronder laten zien hoe.)
    1. Selecteer Sprite(2) door op zijn knop te klikken onder het speelveld.
    2. Vervang het herhaal blok met een herhaal tot blok.
    3. Sleep in de zeshoekige ruimte in het herhaal tot blok. Stel het blok in zodat het blijft herhalen totdat het Sprite aanraakt.
  2. Doe hetzelfde voor Sprite. Vervang de code zodat hij alleen je muis volgt totdat hij Sprite(2) aanraakt.
  3. Test je programma om te kijken of het doet wat je wil:
    • Als je klikt op , zou Sprite (de grijze) je muis moeten volgen en Sprite(2) zou naar Sprite moeten wijzen.
    • Als je Sprite voldoende dichtbij Sprite(2) zet, zouden beide Sprites moeten stoppen met bewegen.
  4. Tot zover wijst Sprite(2) naar Sprite, maar hij beweegt niet naar Sprite toe. Voeg een regel code toe aan het script van Sprite(2) zodat hij Sprite achtervolgt. Dit is het idee:
    Vul het invoerveld in het neem _ stappen blok met een laag getal (waarschijnlijk minder dan 1) zodat Sprite(2) niet te snel bij Sprite is.
  5. Test je programma een paar keer.

  6. Op dit moment, als de sprites elkaar tegenkomen, stoppen ze alleen maar. Zorg dat ze een praatje maken wanneer ze stoppen. Dit kan je doen door code toe te voegen aan het script van Sprite(2). Bedenk zelf wat ze moeten zeggen, je kan iedere taal gebruiken die je kan typen.

    Sprite(2) doet dingen zelf. Hij vertelt ook aan Sprite wanneer hij wat moet doen. Deze code geeft Sprite(2) de leiding over Sprite.
Om te voorkomen dat de twee sprites vast zitten tegen elkaar, niet in staat te bewegen, hebben de sprites wat ruimte nodig tussen elkaar en wat tijd los van elkaar voordat de achtervolging begint. Blokken zoals deze kunnen hierbij van pas komen.
  1. Geef de sprites een ander uiterlijk.
    Je kan een plaatje van het internet gebruiken door het plaatje in het Uiterlijken-tabblad te slepen van de Sprite.
  2. Verander de achtergrond van het speelveld.
