Het delen van je App

Op deze pagina, leer je hoe je app kunt delen middels een link en hoe je je app kunt laden op een mobiel apparaat.
  1. Zorg ervoor dat je project opgeslagen is.
  2. Kies "Open..." van het bestandmenu en selecteer het bestand van je project lijst (In deze les, "H1L1-Alonzo"), en klik "Share." (Zie de video hieronder.)
    3. Om je bestanden snel te vinden, type "H1" in je zoekbalk om alleen een lijst van Hoofdstuk 1-bestanden te zien.
  3. Je wordt gevraagd of je zeker weet dat je het wilt publiceren. Klik op "Ja." (Daarna "Open" of "Annuleren" om het menu de sluiten. Het maakt niet uit op welke je klikt.)
  4. Kopieer de URL (webadres) boven aan het browser venster. Je kan deze link gebruiken om je app te delen.
    5. Je kan ook een korte URL maken voor je app om het typen van de URL makkelijker te maken.

    Let op: Dit zijn kopieën—niet dezelfde bestanden. Dus, wanneer je het project verandert op het ene account zal deze niet veranderen op het andere account; je moet het opnieuw opslaan op het andere account.
Om Alonzo te kunnen bedienen op een mobiel apparaat, moet je zorgen dat de optie "versleepbaar" boven Scripts niet is aangevinkt.
  1. Laad de app op een mobiel apparaat:
    • Op Apple iOS...
      1. Op het mobiele apparaat, open de project URL van stap 3 of 4 hierboven in Safari.
      2. Bovenaan het Safari scherm, druk op het deel icoon, selecteer dan "Toevoegen aan beginscherm" van het menu dat verschijnt.
    • Op Android...
      1. Op het mobiele apparaat, open de project URL van stap 3 of 4 hierboven in Chrome.
      2. Druk op het ☰ instellingen icoon, selecteer daarna "Toevoegen aan beginscherm" van het menu dat verschijnt.
  2. via de email, SMS, of sociale media door de URL te zenden naar je vrienden!
